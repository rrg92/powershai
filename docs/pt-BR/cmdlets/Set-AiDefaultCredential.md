# Set-AiDefaultCredential

## SYNOPSIS <!--!= @#Synop !-->
Define a credential default, que deve ser usada quando o provider precisa autenticar!

## SYNTAX <!--!= @#Syntax !-->

```
Set-AiDefaultCredential [[-Credential] <Object>] [<CommonParameters>]
```
