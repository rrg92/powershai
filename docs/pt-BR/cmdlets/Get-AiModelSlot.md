# Get-AiModelSlot

## SYNOPSIS <!--!= @#Synop !-->
Obtém o slot de model settings. Deve ser usado por usuário avançados para alterar configurações de modelos!

## SYNTAX <!--!= @#Syntax !-->

```
Get-AiModelSlot [[-ModelName] <Object>] [<CommonParameters>]
```

## PARAMETERS <!--!= @#Params !-->

### -ModelName
nome do modelo
