# Set-OllamaUrl

## SYNOPSIS <!--!= @#Synop !-->
Muda a URL do ollama. Formato protocolo://endereço:porta

## SYNTAX <!--!= @#Syntax !-->

```
Set-OllamaUrl [[-url] <Object>] [<CommonParameters>]
```
