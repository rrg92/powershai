# Enable-AiScreenshots

## SYNOPSIS <!--!= @#Synop !-->
מפעיל את מסך ההסבר!

## DESCRIPTION <!--!= @#Desc !-->
מסך ההסבר הוא תכונה המאפשרת לקבל צילומי מסך של אזור מהמסך ולשלוח ל-LLM שתומך בראיה!
זו תכונה שנמצאת עדיין בשלב הניסוי, ולכן יש להפעיל אותה!

## SYNTAX <!--!= @#Syntax !-->

```
Enable-AiScreenshots [<CommonParameters>]
```
